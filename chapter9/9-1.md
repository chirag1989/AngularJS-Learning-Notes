在 `angular.bootstrap()` 时也可以定义：

```html
  angular.bootstrap(document.documentElement, [  
    function($routeProvider){  
	  $routeProvider.when('/test',  
	    {  
		  template: 'test'
		}
      );
	}
  ]);  
```
